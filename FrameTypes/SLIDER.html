FrameType SLIDER

Slider is an interactive frames to set a value inside the bounds of a min and max. Min and max are valid values. The user is able to set the value by draging the button of the Slider or clicking inside the Slider. Warcraft 3 has Vertical or Horizontal Sliders.

Mostly used as free. For a functional child SCROLLBAR.

FDF-Actions

  • SliderInitialValue
  • SliderLayoutHorizontal
  • SliderLayoutVertical
  • SliderMaxValue
  • SliderMinValue
  • SliderStepSize
  • SliderThumbButtonFrame
  • ScrollBarDecButtonFrame
  • ScrollBarIncButtonFrame

Supported FrameEvents

  • FRAMEEVENT_CONTROL_CLICK (BlzFrameClick)
  • FRAMEEVENT_MOUSE_ENTER
  • FRAMEEVENT_MOUSE_LEAVE
  • FRAMEEVENT_MOUSE_UP
  • FRAMEEVENT_MOUSE_WHEEL
  • FRAMEEVENT_SLIDER_VALUE_CHANGED
