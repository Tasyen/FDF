FrameType SPRITE

SPRITE is a non-Interactive Frame displaying a Warcraft 3 model and loops it's animation. Changing the Parent of a Sprite during the game can bug, speeding up the displayed animation. Game Pause does not stop the animation

Similiar: MODEL, STATUSBAR
