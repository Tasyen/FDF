FrameType MENU

Container of the choices of a POPUPMENU.

FDF-Actions

  • MenuItem
  • FrameFont
  • MenuTextHighlightColor
  • MenuItemHeight
  • MenuBorder
  • ControlBackdrop

Supported FrameEvents

  • FRAMEEVENT_MOUSE_ENTER
  • FRAMEEVENT_MOUSE_LEAVE
  • FRAMEEVENT_MOUSE_UP
  • FRAMEEVENT_MOUSE_WHEEL
Example 
IncludeFile "UI/FrameDef/UI/EscMenuTemplates.fdf",
Frame "MENU" "TestPopupMenu" INHERITS WITHCHILDREN "EscMenuPopupMenuMenuTemplate" {
    // the selectable options
    // they will try to load a Localized String
    MenuItem "TestA",     -2,
    MenuItem "TestB",     -2,
    MenuItem "TestC",     -2,
    MenuItem "TestD",     -2,
}
