FrameType TEXT

Basic text display of the Frame Family. Can be a oneLine or multiline text.

To have an auto expanding multiline text set Width x and Height 0.

FDF-Actions

  • FontColor
  • FontDisabledColor
  • FontHighlightColor
  • FrameFont
  • FontShadowColor
  • FontShadowOffset
  • FontFlags
  • FontJustificationH
  • FontJustificationV
  • FontJustificationOffset

Supported FrameEvents

  • FRAMEEVENT_CONTROL_CLICK
  • FRAMEEVENT_MOUSE_ENTER
  • FRAMEEVENT_MOUSE_LEAVE
  • FRAMEEVENT_MOUSE_UP
  • FRAMEEVENT_MOUSE_WHEEL
