ToolTip

Format: ToolTip FrameName,

Example: ToolTip "CodeTextFrame",

Purpose: After this Frame was created (Includes all children), "call" BlzFrameSetTooltip(this, BlzGetFrameByName(frameName, createdContext)). When the ToolTipFrame is created in another Frame then the ToolTipFrame needs to be written at an earlier line.

Used in: none SimpleFrames
